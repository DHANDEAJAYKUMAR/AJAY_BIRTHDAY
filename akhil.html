Class loo evaru oka maata matlade mundhu padhiki padhi sarlu alochisthaaru ani nannu adigithe, naaku first gurthukochedhi Akhil hee.
Asalu class antha oka ethu ayithe, Akhil mathram inko ethu.
Endhukante nenu mana class vaalla andharitho matladadaniki try chesthaanu, but every time nenu evaritho communication miss avuthunna ante Akhil thone.
Nijaniki thanu class loo ekkuva matladadu, but okkasari matladadu ante ika volcano vachinatte!
Ante, thanu cool ga vunnantha sepu baagane matladathaadu, but eppudaithe angry vachindho ika thanu thana vishwarupam chupisthaadu.
Thanatho naaku matladaalane vuntundhi, but alanti situation ekkuva raadhu.

Good habits:-
-> Thanaku ee time ki ela matladaalo baaga thelusu.
-> Thanu okkasari emaina work start chesaada ante, disturb avvadu, verevallani disturb cheyadu, and aa work finish chese antha varaku kuda oorukodu.
-> Thanu friendship ki value isthaadu.
-> Face-to-face matlade person.
-> Thappu chesthe oppukuntaadu, but thappu cheyyakapothe thala vanchadu.

Bad habits:-
-> Thanu endhuko class loo konchem silent, so ala kakunda andharitho free ga move ayithe inka better ani naa feeling.

Nick name:- "SILENT STRATEGIST" (for his thoughtful nature and calm demeanor).
