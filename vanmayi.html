Asalu class loo evaru neetho 1 year ki okasari matladathaaru ante naaku first gurthukochedhi Vanmayi.
Nijaniki maa idhariki communication cheyalane vuntundhi but alanti situations raavu.
Thanakunna best skill entante drawing.
Asalu thanu vesina drawing inka evaru veyaleru annanthaga vuntundhi.
Asalu thanu chinnapillalaaga behave chesthundhi, nijaniki thanu ala vundadanike ekkuva ishtapaduthundhi.
Asalu class loo evaru ekkuvaga face expressions isthaaru ante adhi Vanmayi hee.

Good habits:-
-> Thana body language perfect ga vuntundhi.
-> Thanu friendship ki value isthundhi.
-> Thanu silent ga thana pani thanu chesukuntu veluthundhi.
-> Anthe thanu ekkuva matladadhu but ee time ki ela matlaadaalo mathram thanaku baaga thelusu.
-> Anni naake vachu ane behaviour thanaku vundadhu.
-> Ee time ki elanti face expressions ivvaalo kuda thanaku thelusu.
-> Seminar ichetappudu edhaina mistake chebuthe thanu thana expressions thone cover chesthundhi.

Bad habits:-
-> Normal ga manandhariki thelisindhe, class loo Hema, Glory, Vanmayi friendship kanna inka strong bond edhi kanipinchadhu ani,
so naa feeling entante ala vaalathone kakunda thanu andharithonu konchem friendly ga move ayithe inka better ani naa feeling.

But nijam cheppalante thanaku bad habits kanna good habits hee ekkuvaga vunnai.
